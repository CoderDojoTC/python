    Math and Arithmetic¶

    Summary¶

    Python is a powerful calculator. This chapter explores every arithmetic operator: integer division (//), the remainder (%), exponentiation (**), order of operations, negative numbers, and scientific notation. You'll also learn to assign multiple variables at once and use augmented operators like += to make your code shorter.

    This chapter covers the following 10 concepts from the learning graph:

    2. Augmented Assignment Operators
    3. Division Returns Float
    4. Integer Division Operator
    5. Modulo Operator
    6. Exponentiation Operator
    7. Arithmetic Operators
