# Image Processing with Pillow

## Summary

Pillow (PIL) lets Python open, modify, and save images. This chapter introduces
the library, opens an image file, displays it, accesses its pixel data as an
array, resizes and crops images, converts between color modes (RGB, grayscale),
and saves the result. Together with NumPy and matplotlib, Pillow forms the
foundation for the machine learning work in the next two chapters.

## Concepts Covered

This chapter covers the following 7 concepts from the learning graph:

1. PIL Pillow Library Overview
2. Image.open() Open Image
3. image.show() Display Image
4. Pixel Data and Image Arrays
5. Image Resize and Crop
6. Image Color Conversion
7. Saving Images
