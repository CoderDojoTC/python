# Algorithms and Data Structures

## Summary

This chapter introduces computer science fundamentals: abstract data types,
stacks (last-in first-out), queues (first-in first-out), graphs (nodes and edges),
adjacency list representation, and the two most important graph-search algorithms —
Breadth-First Search (BFS) and Depth-First Search (DFS). You'll also cover
sorting algorithms (bubble, selection, insertion), binary search, Big-O notation,
space complexity, linked lists, and trees. The capstone project: a maze solver.
