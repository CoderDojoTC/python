# User Input and Type Conversion

## Summary

`input()` lets your program ask the user a question and read their answer.
But `input()` always returns a *string* — even if the user types a number.
This chapter covers converting between types with `int()`, `float()`, `str()`,
and `bool()`, checking a value's type with `type()`, stripping extra spaces,
validating input, and using `print()` and `input()` as true built-in functions.
