# More Booleans and Conditionals

## Summary

This chapter deepens your understanding of truth in Python: truthiness and
falsiness (any value can be tested), short-circuit evaluation, converting
values to bool, checking types with `type()` and `isinstance()`,
and advanced conditional syntax including ternary expressions,
`match/case` statements, and compound conditions with `and`/`or`.

## Concepts Covered

This chapter covers the following 11 concepts from the learning graph:

1. Truthiness and Falsiness
2. Short-Circuit Evaluation
3. Functions Returning None
4. Checking for None with is
5. Implicit vs Explicit Conversion
6. Reading Multiple Inputs
7. Strip Whitespace from Input
8. Conditional Ternary Expression
9. match/case Statement
10. Truthiness in Conditions
11. Compound Conditions
