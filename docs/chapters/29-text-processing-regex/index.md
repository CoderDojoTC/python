# Text Processing and Regular Expressions

## Summary

Regular expressions (regex) are a mini-language for describing text patterns.
They sound scary but they're incredibly useful for searching, validating,
and transforming text. This chapter introduces the `re` module,
`search()`/`findall()`/`split()`/`sub()`, digit/word/whitespace patterns,
anchors, character classes, quantifiers, alternation, escaping special
characters, raw strings for regex, capturing groups, and compiled patterns.
