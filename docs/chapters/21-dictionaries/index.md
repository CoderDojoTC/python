# Dictionaries

## Summary

A dictionary maps *keys* to *values* — like a real dictionary maps words to
definitions. Dictionaries are one of Python's most useful data structures.
This chapter covers creating dicts, accessing values by key, `get()` for safe
access, adding/updating/removing entries, iterating over keys/values/items,
membership testing, nested dictionaries, and dictionary comprehensions.
