# Colors, Fill, and Turtle Art

## Summary

Colors make turtle art come alive. This chapter goes deep on color:
hex color codes, the RGB color model (red + green + blue = any color),
the HSL model, transparency, color theory basics, and how to set turtle's
pen and background colors. You'll also learn `begin_fill()`/`end_fill()`
to paint solid shapes, creating your most beautiful turtle artwork yet.

## Concepts Covered

This chapter covers the following 8 concepts from the learning graph:

1. Hex Color Values
2. RGB Color Model
3. Color in Turtle Graphics
4. Background and Pen Colors
5. HSL Color Model
6. Color Theory Basics
7. Transparency and Alpha
8. Fill begin_fill() end_fill()
