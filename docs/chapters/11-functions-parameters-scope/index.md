# Functions with Parameters and Scope

## Summary

Functions become much more powerful when you give them *parameters* (inputs).
This chapter covers default parameter values, keyword arguments, what happens
to variables *inside* vs *outside* functions (scope), the `global` keyword,
and how to use `nonlocal` in nested functions.
You'll also practice writing pure functions that return useful values.
