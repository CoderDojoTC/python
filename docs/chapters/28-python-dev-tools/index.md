# Python Development Tools

## Summary

Skulpt and Trinket are great for learning, but real Python projects use richer
tools. This chapter surveys the Python ecosystem: Repl.it, Thonny (a beginner
desktop IDE), Spyder (scientific IDE), VS Code (professional editor), Jupyter
Notebooks and JupyterLab, Conda and pip for package management, virtual
environments, installing Python locally, running scripts from the terminal,
Raspberry Pi, Google Colab, and Python 2 vs Python 3.
