# Advanced Functions and OOP

## Summary

This chapter rounds out your functional-programming and OOP knowledge.
You'll learn `map()` and `filter()` for transforming collections, `dir()` and
`help()` for exploring any object, `id()` and `hash()` for object identity,
`any()`/`all()` for aggregate testing, and `chr()`/`ord()` for character math.
On the OOP side: exception hierarchy, custom exception classes, exception
chaining, static methods, `@property`, dunder methods, and object composition.

## Concepts Covered
