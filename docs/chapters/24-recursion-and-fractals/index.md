# Recursion and Fractals

## Summary

Recursion is when a function calls *itself* — a mind-bending but powerful idea.
This chapter introduces the `nonlocal` keyword, the recursion concept, base cases
(to stop infinite loops), call-stack depth limits, lambda (one-line) functions,
functions as arguments, and nested functions. The payoff: a beautiful
recursive **fractal tree** drawn with turtle graphics.

## Concepts Covered

This chapter covers the following 8 concepts from the learning graph:

1. nonlocal Keyword
2. Recursion Concept
3. Recursion Base Case
4. Recursion Depth and Stack
5. Lambda Functions
6. Function as Argument
7. Nested Functions
8. Fractals with Recursion and Turtle
