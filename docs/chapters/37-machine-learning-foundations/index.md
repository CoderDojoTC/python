# Machine Learning Foundations

## Summary

Machine learning (ML) lets programs *learn from data* instead of following
hand-written rules. This chapter introduces the big picture — supervised vs
unsupervised learning — then moves to the Keras library to build your first
neural network. You'll learn the sequential model, dense (fully connected) layers,
activation functions (relu, softmax), loss functions, the Adam optimizer,
accuracy metrics, batch size and epochs, and the famous MNIST handwritten-digit
dataset, including data normalization and the train/validation/test split.
