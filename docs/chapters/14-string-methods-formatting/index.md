# String Methods and Formatting

## Summary

Strings have dozens of useful built-in methods. This chapter covers indexing
individual characters, slicing substrings, `split()`/`join()`/`replace()`,
`startswith()`/`endswith()`/`find()`, the powerful `format()` method,
**f-strings** (the modern way to embed variables in text), string comparison,
and multiline string formatting for tidy output.

## Concepts Covered

This chapter covers the following 9 concepts from the learning graph:

1. String Indexing
2. String Slicing
3. String split() join() replace()
4. String startswith() endswith() find()
5. String isdigit() isalpha() isalnum()
6. String format() Method
7. f-Strings
8. String Comparison
9. Multiline String Formatting
