# Lists — Methods and Patterns

## Summary

Lists come with powerful built-in methods for adding, removing, and reorganizing
items. This chapter covers `append()`/`insert()`/`extend()`, `remove()`/`pop()`,
`sort()`/`reverse()`, iterating over lists with `for`, nested lists, list
comprehensions, the loop-with-list-building pattern, and using lists as
stacks (LIFO) and queues (FIFO).

## Concepts Covered

4. List sort() reverse() copy()
5. List index() count()
6. List Concatenation and Repetition
7. Iterating over a List

11. List as Queue
