# Lists — Basics

## Summary

A list is an ordered collection of items — like a Scratch list block but much
more powerful. This chapter introduces the list type, how to create lists,
access items by index, use negative indexing to count from the end, slice
sub-lists, measure length with `len()`, test membership with `in`,
and understand the key difference between mutable and immutable types.

## Concepts Covered

This chapter covers the following 8 concepts from the learning graph:

1. List Type
2. Creating a List
3. List Indexing
4. Negative Indexing
5. List Slicing
6. len() for Lists
7. Membership with in Operator
8. Mutable vs Immutable Types
