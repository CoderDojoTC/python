# Math Module and Turtle Projects

## Summary

The `math` module unlocks trigonometry, logarithms, and constants like `pi`.
This chapter covers `abs()`, `round()`, `sqrt()`/`floor()`/`ceil()`,
`sin()`/`cos()`/`pi`, radians/degrees conversion, and more.
You'll build two satisfying turtle projects: a **sine wave** that ripples across
the screen, and a **turtle race** where randomness decides the winner.
