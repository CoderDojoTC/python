# Advanced Turtle and Event-Driven Programming

## Summary

Programs that respond to mouse clicks and key presses use *event-driven*
programming. This chapter covers the event-driven model, mouse-click and
keyboard event handlers, callback functions, animation loops, the event queue,
timer-based events, and three advanced turtle projects: an interactive color
picker (`onscreenclick()`), a maze visualization, and a color-bars demo.
Jupyter Canvas users get `ipycanvas` and `mobilechelonian` too.

## Concepts Covered

1. onscreenclick() Event Handler
2. Color Picker with Turtle
3. Maze Visualization with Turtle
4. ipycanvas Canvas in Jupyter
