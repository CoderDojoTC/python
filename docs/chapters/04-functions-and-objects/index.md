# Functions and Objects

## Summary

Functions let you name a group of steps so you can use them again and again —
just like a custom Scratch block. This chapter teaches you to define functions
with `def`, call them, give them parameters, and return values.
You'll also get a friendly first look at Python *objects* (like the turtle you'll
meet in the next chapter) and explore the Standard Library of built-in tools.

## Concepts Covered

This chapter covers the following 8 concepts from the learning graph:

1. Defining a Function with def
2. Calling a Function
3. Positional Parameters
4. return Statement
5. Functions Returning None
6. Docstrings
7. Standard Library Overview
8. Objects and Classes Overview
