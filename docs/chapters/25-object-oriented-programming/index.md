# Object-Oriented Programming

## Summary

Object-Oriented Programming (OOP) is how Python organizes complex programs.
You've already used objects (the turtle!); now you'll create your own.
This chapter covers attributes, methods, the `class` keyword, `__init__()`,
`self`, creating instances, dot notation, OOP in strings and lists,
inheritance, `__str__()`/`__repr__()`, subclasses, `super()`,
and class vs instance variables.
