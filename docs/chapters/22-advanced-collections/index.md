# Advanced Collections and Built-in Functions

## Summary

Python's built-in function library is huge. This chapter collects the most
useful ones: `isinstance()`, `enumerate()` for indexed loops, `zip()` to combine
iterables, `*args`/`**kwargs` for flexible functions, returning multiple values,
named tuples, building dicts from two lists, `len()` for sequences, `range()`,
and the core aggregating functions: `max()`/`min()`/`sum()`,
`sorted()`/`reversed()`, `map()`/`filter()`.
