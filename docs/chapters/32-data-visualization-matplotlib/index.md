# Data Visualization with matplotlib

## Summary

matplotlib is Python's most popular charting library. This chapter covers the
full workflow: importing `pyplot as plt`, creating line plots, displaying them
with `show()`, adding titles and axis labels, drawing grids, scatter plots,
bar charts, histograms, legends, saving figures to disk, plotting multiple
lines on one chart, and organizing multiple charts with subplots.

## Concepts Covered

This chapter covers the following 14 concepts from the learning graph:

1. matplotlib Library Overview
2. import matplotlib.pyplot as plt
3. plt.plot() Line Plot
4. plt.show() Display
5. Plot Title and Axis Labels
6. plt.grid() Grid Lines
7. Lists as XY Plot Data
8. plt.scatter() Scatter Plot
9. plt.bar() Bar Chart
10. plt.hist() Histogram
11. plt.legend() Legend
12. plt.savefig() Save Plot
13. Multiple Lines on One Plot
14. Subplots with plt.subplot()
