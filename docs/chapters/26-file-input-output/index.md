# File Input and Output

## Summary

Programs become much more useful when they can read and write files — saving
scores, loading data, processing text. This chapter introduces the command line,
navigating the file system, `open()`, file modes (`r`, `w`, `a`), all the read
and write methods, the `with` statement for safe file handling, iterating over
lines, text processing with `strip()`/`lower()`, and reading and writing CSV files.
