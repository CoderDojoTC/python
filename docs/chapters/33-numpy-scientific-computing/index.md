# NumPy and Scientific Computing

## Summary

NumPy is the foundation of scientific Python. Its *array* type is like a list
but blazing fast and capable of math on every element at once. This chapter
covers importing numpy, creating arrays, element-wise operations and broadcasting,
reshaping, `zeros()`/`ones()`, `linspace()` for evenly spaced values, array
slicing, `mean()`/`std()`, argmax, and plotting a sine wave with matplotlib.
