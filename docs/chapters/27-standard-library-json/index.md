# Standard Library and JSON

## Summary

Python ships with a huge standard library of ready-to-use modules.
This chapter covers creating your own custom modules (and the `__main__` guard),
the `string` module, `sys`, `os` and `os.path`, the `json` module for reading
and writing structured data, `csv`, `datetime`, `collections`, `itertools`,
reading/writing JSON files, and working with file paths and checking existence.
