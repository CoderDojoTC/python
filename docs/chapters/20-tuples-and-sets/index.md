# Tuples and Sets

## Summary

Python has two more collection types beyond lists. **Tuples** are like lists
but immutable (unchangeable after creation) — perfect for coordinates, RGB triples,
and function return values. **Sets** store unordered unique items and support
fast membership testing plus mathematical operations: union, intersection,
and difference. This chapter covers both fully.
