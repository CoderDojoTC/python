## turtle graphics
The following code is an example of drawing a square using turtle graphics.

```
import turtle
dan = turtle.Turtle()
dan.shape('turtle')

dan.forward(40)
dan.right(90)

dan.forward(40)
dan.right(90)

dan.forward(40)
dan.right(90)

dan.forward(40)
dan.right(90)
   
dan.write('done with square')
```
